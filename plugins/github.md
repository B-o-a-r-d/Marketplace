The **GitHub** Power-Up connects a board to a repository.

- **Special lists** fed by a repo's recent **commits**, open **pull requests** or
  open **issues** — read-only, lazy-loaded, refreshed live.
- **Card enrichment** — attach a commit / PR / issue to a card (paste a URL or
  `owner/repo@sha`); the status renders on the card and a GitHub activity is logged.
- **MCP tool** — list a repository's recent commits from an AI agent.

Requires connecting a GitHub OAuth app (configured per board, credentials stored
encrypted).
